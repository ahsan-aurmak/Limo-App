# Limo App MVP

Compliance-first web app foundation for Abu Dhabi LES trip registration.

## Stack

- React + TypeScript + Vite
- Ant Design v5

## Run

```bash
npm install
npm run dev
```

## Next MVP modules

- LES auth and connector service
- Driver and vehicle permit sync
- Trip validation and submission queue
- Failed submission reconciliation
