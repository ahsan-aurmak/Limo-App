# Carmak Driver App

Separate Expo React Native app for private/manual limousine bookings captured by drivers.

## Preview

1. Open terminal in `mobile-driver-app`
2. Run `npm install`
3. Run `npm start`
4. Scan the Expo QR code using Expo Go on Android, or press `w` for browser preview

## Demo Login

- Username: `driver`
- Password: `Carmak@2026`

## Current Scope

- login
- today bookings
- booking detail
- passenger verification
- new private booking
- sync status
- profile/logout

This is intentionally separate from the main web app and is currently using mocked data and mocked sync behavior that aligns with the agreed driver-app contract.
